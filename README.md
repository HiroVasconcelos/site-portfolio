# Portfólio | Guilherme Ayres Vasconcelos

Repositório do meu portfólio, desenvolvido para centralizar e expor meus projetos pessoais e informações profissionais.

🔗 **[Acessar o Portfólio Online](https://portfolio-guilherme-ayres-vasconcelos.vercel.app/)**

## 📌 Sobre o Projeto
O objetivo deste projeto foi construir uma página web responsiva e com alta performance, sem a complexidade desnecessária de frameworks JavaScript pesadas (como React ou Angular).

## 🛠️ Stack Tecnológica
* **HTML5:** Estrutura semântica.
* **Tailwind CSS:** Estilização utilitária para prototipagem rápida e design responsivo.
* **JavaScript:** Lógica de interface (Intersection Observer para animações de scroll, manipulação do DOM para o menu mobile e sistema de Toast Notifications).
* **Lucide Icons / SVG:** Ícones vetoriais.
* **Formspree:** Serviço de form-to-email para processar mensagens de contato sem necessidade de um backend dedicado.
* **Vercel:** Hospedagem e pipeline de CI/CD para deploy automático via GitHub.
